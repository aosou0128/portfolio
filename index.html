web制作について

侍テラコヤというサブスク型プログラミングスクールに登録し、
webデザインの基礎からコーディングの基礎まで勉強しつつ制作を進めました。

制作をしようと思ったきっかけは、iPhoneのwebページを閲覧した際にスクロールと共に動く文字や画像などのアニメーションに感銘を受け、自分で作ってみたいと思ったためです。
元々コードを入力して何かを開発することに興味があったため、実際に制作に挑戦してみたいと思いすぐに勉強を始めました。

  • ①webデザインカンプ作成

    adobe XDを使用し、文字や線を配置しつつ大きさや色を決めていきました。
    直感的に配置をしたり動きを付けられてとても便利だと思ったため、すぐに使用方法を勉強し作成を進めました。
    iPhoneの紹介ページを参考に作成しております。

  • ②コーディング

    デザインカンプ通りに表示できるよう、言語の勉強をしつつ実際にコードを入力して進めました。

  • ○HTML

    web制作の基礎となる言語で、実際にwebページに表示する文字の記述だけでなく、
    metaタグという検索エンジンやブラウザにwebページの情報を伝えるための設定やCSSなどの他のファイルを読み込むための設定を記述しております。

    See the Pen Untitled by aosou0128 (@aosou0128) on CodePen.

  • ○CSS

    HTMLに記述した実際にwebページに表示する文字を指定し、色や大きさやフォントなどの見た目の設定を記述しております。

    See the Pen CSS by aosou0128 (@aosou0128) on CodePen.

  • ○SCSS

    SassというCSSを簡単に記述できる言語でSCSSという記法があり、記述を効率化できるため取り入れました。
    「変数やmixin(頻繁に使用するCSSの設定を決めておき簡単に使い回せる)を使用できるため記述を簡略化できる」
    「ネスト構造(入れ子)にできるためシンプルな記述でどの文字を指定しているか分かりやすい」
    「項目ごとにファイルを分けて記述してまとめることができるため管理を行いやすい」というメリットがあるため、記述方法や導入方法を調べて取り入れました。

    See the Pen Untitled by aosou0128 (@aosou0128) on CodePen.

  • ○js

    jQueryという、webブラウザ用のJavaScriptコードを簡単に記述できるように作られたライブラリを取り入れました。
    表示している部分の目次をハイライト表示させたり、目次をクリックするとスムーズに移動させたり、文字のアニメーションを付けるためのプログラムを記述しております。
    表示するデバイスによって動画の埋め込みのサイズを自動調節するプログラムも記述しており、理解を深めていきたい言語です。

    See the Pen Untitled by aosou0128 (@aosou0128) on CodePen.

  • ○img(画像ファイル)

    webページの中で使用している画像を格納しております。
